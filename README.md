# Travel agency webpage.

Travel agency webpage - work in progress.

# Technologies used

- Nuxt3
- Taiwlind
- Strapi
- Typescript
- AOS

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert supabase connection.

```bash
cp .env.example .env
```
